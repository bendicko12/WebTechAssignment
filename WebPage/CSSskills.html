What is CSS?

CSS (Cascading Style Sheet) is a tool used to develop, style and design to web pages. The basics of a web page are written in HTML and provide the basic outline and structure, then the CSS is applied to manipulate the HTML into an aesthetically pleasing and functional page. CSS is just extra code that tells the web page a variety of things to do such as; change colour, shape, alignment, images, sizing, borders, create tables etc. Below are 2 separate designs of a pet web page, both have the same HTML and have only been altered using CSS, one for the ages 16+ and the other for children aged 7-9.
